# Canvas - Development Center

## Description 

Canvas is a versatile web app that will allow me to research, design, develop, implement and design software applications.


## Thoughts

1. Canvas will let me "spitball" and develop software application ideas.
2. Canvas will very easily let me turn ideas into software projects.
3. Canvas will help me architect and design software.
4. Canvas will give me guidance and direction when using tools like VS Code.
5. Canvas will have a wealth of APIs that other applications can use to retrieve information when beginning task and when ending a task.
6. Canvas will have a "long term memory" capability where it can store and retrieve information. The information should be relatable to other "memory" information and create something like a memory graph. 
7. The "long term memory" system will function like a Retrieval-Argument Generation (RAG) system.
